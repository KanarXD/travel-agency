# travel-agency

### requirements:

* java 16
* node.js
* docker

### database:

* run `docker-compose up -d`

### server-app:

* http://localhost:4000

### web-app:

* download dependencies `npm install`
* run `ng serve`
* http://localhost:5000

### phppgadmin:

* http://localhost:1080
* login: postgres
* password: root
